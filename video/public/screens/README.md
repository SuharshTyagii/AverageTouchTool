# Screenshots for the promo

Drop real PNG screenshots of the app here, then reference them in
`src/scenes/Screens.tsx` by setting the `file` fields, e.g.:

```ts
const SHOTS = [
  { file: "settings.png", caption: "Settings · profiles & bindings" },
  { file: "action-picker.png", caption: "Action picker · searchable" },
];
```

Suggested shots (open the app → ⌘, for Settings):
- `settings.png` — the three-pane Settings window on the Global profile
- `action-picker.png` — the binding editor with the categorized action popover open
- `touchbar.png` — the Touch Bar modal (optional)

Tip: capture a window cleanly with `⌘⇧4` then Space, or use AverageTouchTool's
own Capture action. Aim for a 16:10-ish crop.
