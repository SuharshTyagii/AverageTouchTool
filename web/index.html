What it reads

Real finger counts, from raw multitouch frames.

macOS won't tell ordinary apps how many fingers are down. AverageTouchTool reads the raw trackpad contact frames directly, so a three-finger swipe is genuinely three fingers — not a guess.

  • Taps

    2, 3, 4 or 5 fingers. Stationary and quick — measured only while every finger is down.

  • Swipes

    Up, down, left, right with 2–4 fingers. Direction from the dominant travel axis.

  • Pinch

    Two fingers, in or out. Classified by change in spread between contacts.

  • Rotate

    Two fingers, clockwise or counter. Angle folded to ±90° so finger order can't flip it.

Don't want to remember the picker? Hit Record in the binding editor and just perform the gesture — it fills in the trigger.

What it does

A searchable action library, by category.

Bind any trigger to one action or a whole chain. Pick from a categorized, searchable list — every entry has an icon.

Get it

Download and drag to Applications.

Grab the latest .dmg, or build from source. No Electron, no runtime — a native menu-bar app.

Download AverageTouchTool.dmg All releases & notes →

First launch: right-click the app → Open (it's signed but not yet notarized, so Gatekeeper asks once). Then grant Accessibility + Input Monitoring (and Screen Recording for captures).

Or build from source:

# clone & build the signed .app
git clone https://github.com/SuharshTyagii/AverageTouchTool.git
cd AverageTouchTool
./package.sh   # builds, signs, installs, launches
